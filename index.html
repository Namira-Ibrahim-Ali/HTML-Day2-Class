What is HTML?

HTML, or Hyper Text Markup Language, is not a scripting language, its markup language!

Top 3 Fun Facts About HTML:

  1. HTML was created by Tim Berners-Lee in 1991.
  2. HTML is the skeleton of every webpage you see on the internet.
  3. HTML introduced the <article> and <section> tags for better semantics

Some Amazing Facts:

  • The <marquee> tag is now obselete but was once a fun way to create scrolling text.
  • The <blink> tag in Netscape used to make a text flash (thankfully it's gone!)
